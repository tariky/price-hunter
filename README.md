## Kako pokrenuti bot

1. Instalirajte potrebne module, sa npm install ili yarn add.
2. Preimenujte config-sample.mjs u config.mjs.
3. Unutar config.mjs datoteke izmjenite zeljene parametre.
4. Pokrenite app sa npm start.
